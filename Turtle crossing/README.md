# Turtle Crossing Game 🐢🚗

A fun **Turtle Crossing Game** built with Python’s `turtle` module. The player controls a turtle trying to cross a busy road without getting hit by cars. The game gets progressively harder as levels increase.

---

## Features

- **Player Movement**: Control the turtle using arrow keys (`Up`, `Down`, `Left`, `Right`).
- **Dynamic Cars**: Cars spawn randomly and move across the screen at increasing speeds.
- **Levels**: Each successful crossing increases the level and speeds up the cars.
- **Collision Detection**: Game ends if the turtle collides with a car.
- **Scoreboard**: Tracks and displays the current level, and shows "Game Over" when the player loses.
